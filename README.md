# Employee-attrition-survival-analysis
This project uses a Cox Proportional Hazards Model to perform survival analysis on employee data, predicting the likelihood of employees staying in an organization over time.  Unlike traditional classification models, this approach focuses on time-to-event prediction, helping organizations understand not just who may leave, but when.

"Predicts not just whether an employee will leave, but estimates the probability of retention over time using survival analysis."
